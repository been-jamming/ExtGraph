Comment utiliser la librairie dans vos programmes ?


  • Si vous programmez en C ou ASM avec TIGCC

    • En utilisant l'IDE
      Ajoutez les fichiers TileMap.a et TileMap.h à votre projet (Project -> Add Files...), puis ajoutez la ligne
      #include "TileMap.h"
      dans chaque fichier source où vous désirez utiliser des fonctions du TileMap Engine.

    • En utilisant une ligne de commandes
      Dans chaque fichier source qui utilise des fonctions du TileMap Engine, incluez le header TileMap.h, en précisant soit le chemin absolu, soit le chemin relatif ; par exemple :
      #include "../../TileMap Engine/Lib/TileMap.h"
      Puis, pour compiler, indiquez comme d'habitude les fichiers source à compiler, suivis de TileMap.a, en précisant là encore soit le chemin absolu, soit le chemin relatif pour y accéder ; par exemple :
      tigcc prog.c sprites.c "../../TileMap Engine/Lib/TileMap.a"
