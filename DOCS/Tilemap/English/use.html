How to use the library in your programs ?


  • If you program in C or ASM using TIGCC

    • Using the IDE
      Add the files TileMap.a and TileMap.h at your project (Project -> Add Files...), and add the line
      #include "TileMap.h"
      in each source file using TileMap Engine's functions.

    • Using a commande line
      In each source file using TileMap Engine's functions, include the header TileMap.h, giving either its relative path, either its absolute path ; for example :
      #include "../../TileMap Engine/Lib/TileMap.h"
      To compile your program, indicate as asual the sources files you want to compile, followed by TileMap.a, giving either its relative path, either its absolute path ; for example :
      tigcc prog.c sprites.c "../../TileMap Engine/Lib/TileMap.a"
